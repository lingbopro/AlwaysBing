## 概述
这个用户脚本用于在打开非必应的搜索页面时自动重定向到必应，或者显示一条通知。  

## 安装

## 使用
首先，确保此脚本为启用状态。  
在必应页面上，你可以在 Tampermonkey 的菜单中更改配置。  
对于一个搜索引擎的配置有：  
- `🔗重定向`: 自动重定向到必应，不提示
- `💬提示`: 显示一个提示（不可关闭）
- `💬提示（可关闭）`: 显示一个提示（可关闭）
- `❌什么也不做`: 顾名思义，什么也不做
